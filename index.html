GitHub Tools

⚙️ Settings · no token

Simple browser-based utilities for auditing your GitHub repos — no backend, no installs. Uses the GitHub API directly with an optional personal access token.

🐛
Issue Auditor
Scan all repos for open issues and pull requests. See tagged vs. untagged issues at a glance, filter to repos that need attention, and jump straight to GitHub.
Issues Pull Requests Labels
🌿
Branch Auditor
View the default branch of every repo across your account. Spot repos still on master, filter by branch name, and check last-push dates.
Branches Default branch
🕸️
Network Graph
Interactive commit and branch history for any repo. Zoom, pan, and hover commits for details — without the bugs in GitHub's own graph.
Commits Branches History
